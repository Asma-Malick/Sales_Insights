# Sales Insights Dashboard (Tableau)

## Overview

This interactive dashboard analyzes sales performance across markets, customers, and products using **Tableau**. It provides a clear view of revenue distribution, sales volume, and trends over time.

## Live Dashboard

[View Dashboard](https://public.tableau.com/views/Sales_Insights_Project_17741759059740/SalesInsights?:language=en-GB&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

## Key Insights

* Total revenue is **414M+**, with **Brick & Mortar contributing the majority** over E-commerce.
* **Delhi NCR dominates** revenue (~221M) and sales quantity, followed by Mumbai and Ahmedabad.
* Revenue shows **fluctuations across months**, with peaks around early and mid-year.
* A small group of customers (Top N) contributes a **significant share of total revenue**.
* Product-level analysis highlights **top-performing products driving sales**.

## Features

* KPI tracking (Revenue and Sales Quantity)
* Market-wise and customer-wise performance analysis
* Monthly revenue trend visualization
* Top N filtering for customers and products

## Tools

* Tableau
* MySQL

## Workflow        

1. Data extraction from MySQL
2. Data transformation and aggregation
3. Dashboard design and visualization
4. Publishing on Tableau Public
